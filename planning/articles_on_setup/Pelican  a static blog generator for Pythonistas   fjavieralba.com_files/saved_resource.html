                                F. Javier Alba Mod Koen Colpaert • a year ago

                                Hi Koen.

                                The pelican docs (http://pelican.readthedocs.org... are quite detailed in how to get it up and running.

                                In order to manage themes, there is a very helpful tool called "pelican-themes" (http://pelican.readthedocs.org....

                                Once you've installed a theme with the "pelican-themes" command, you can tell pelican to use it when generating your blog files with the -t option. For example:

                                pelican my_rst_dir/ -s conf.py -t my_theme

                                You can also specify a THEME variable inside your conf.py

                                hope it helps
